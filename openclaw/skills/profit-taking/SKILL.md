# Profit Taking Skill — 止盈行动计划

## 触发条件

### 定时触发
- Cloud Scheduler / OpenClaw cron：工作日 09:00。
- 用途：开盘前半小时扫描所有活跃持仓，生成当天可能触达止盈目标的操作建议。

### 手动触发
用户通过微信 clawbot 发送：
- "止盈计划"
- "今日止盈"
- "止盈建议"

## 输入格式

### Cron
```json
{
  "trigger_type": "cron",
  "plan_date": "2026-05-08",
  "user_scope": "all_active_positions"
}
```

### 手动
```
用户：今日止盈
用户：止盈建议 AAPL
```

## 处理逻辑

1. 拉取所有 `position_snapshots.total_quantity > 0` 的最新持仓。
2. 获取持仓个股最近一年日线历史数据，以及所属市场的大盘指数历史数据。
3. 回测候选止盈规则：
   - 固定浮盈阈值：10% / 15% / 20%
   - ATR 动态保护：2.0 / 2.5 / 3.0 倍 ATR
   - 20 日高点回撤保护：4.5% / 6% / 7.5%
   - 规则有效要求：样本数不少于 4，胜率不低于 50%。
4. 识别大盘状态：
   - `risk_off`：指数低于 20 日均线且 20 日跌幅超过 3%，降低止盈阈值。
   - `risk_on`：指数在 20/60 日均线上方且 20 日趋势向上，提高止盈阈值。
   - `neutral`：沿用回测选出的阈值。
5. 跟踪个股关键指标：
   - 当前浮盈
   - ATR14
   - RSI14
   - 20/60 日均线
   - 20 日高点回撤
   - 今日触达目标价概率：`low` / `medium` / `high`
6. 生成行动：
   - `TAKE_PROFIT`：浮盈达到阈值，且出现超买、回撤或跌破短期趋势。
   - `WATCH_TARGET`：尚未触发，但目标价与现价距离小于 1 日 ATR，今日可能触达。
   - `HOLD`：未触发或回测无效。
7. 写入 `profit_taking_plans`，对 `TAKE_PROFIT` / `WATCH_TARGET` 创建 `delivery_runs` 待推送。

## 输出格式

### 止盈行动
```
止盈提醒：贵州茅台(SH600519) 已触发止盈纪律。可考虑分批降低约 33% 仓位（约 100 股），参考目标价 1850.00，动态保护价 1790.00。依据：当前浮盈达到回测验证阈值，RSI 偏高且从 20 日高点回撤扩大。执行前请结合实时盘口和账户情况确认。
```

### 止盈观察
```
止盈观察：AAPL(AAPL) 今日接近止盈观察价 212.00。若盘中放量冲高且接近该价位，可优先评估是否分批止盈；动态保护价 204.50。
```

## 数据库操作

- `position_snapshots`: SELECT 最新活跃持仓
- `user_sessions`: SELECT 最新微信会话
- `profit_taking_plans`: INSERT / UPDATE 每日止盈行动计划
- `delivery_runs`: INSERT 待推送操作建议
- `job_runs`: 关联定时任务运行记录

## 风险边界

- 规则必须经过历史数据回测验证后才会发出 `TAKE_PROFIT` 或 `WATCH_TARGET`。
- 无法获取历史数据、持仓成本或当前价格时，只生成 `HOLD` 或跳过该持仓。
- 推送内容是风险提醒和纪律建议，不直接下单。
